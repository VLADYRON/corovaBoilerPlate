Libraries
--------

It's a boilerplate for Html5 Mobile apps using Cordova, Backbone 1.2.3, Twitter Bootstrap 3.3.5, underscore 1.8.3

Usage
----

git clone
cordova platform add ios
cordova prepare ios
cordova run ios
